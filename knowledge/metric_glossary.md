# Metric Glossary Index | 指标字典索引

这个文件是 **formal glossary 的索引入口**，不是单独承载所有正式指标条目的大文件。  
This file is the **index entry for the formal glossary**, not a single giant file that stores every formal metric entry.

正式 glossary 请按层读取：

1. 指标组索引  
   `knowledge/glossary/metric_groups.md`
2. 维度索引  
   `knowledge/glossary/dimensions.md`
3. 按指标组拆分的正式指标文件  
   `knowledge/glossary/metrics_by_group/*.md`
4. 如有需要，再读取维度值索引  
   `knowledge/glossary/dimension_values.md`

填写规则、字段解释、模板和 draft/formal 流转说明，请看：

- `knowledge/metric_glossary_guide.md`

## Read Order | 推荐读取顺序

如果当前任务是在分析实验，推荐按这个顺序取 glossary：

1. 先看指标组是什么  
2. 再看该组里的正式指标  
3. 最后看相关维度定义
